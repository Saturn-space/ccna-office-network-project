# 🖧 CCNA Networking Projects

## 📌 Overview
This repository contains a collection of my hands-on networking projects built using Cisco Packet Tracer as part of my CCNA learning journey.

These projects demonstrate my understanding of networking concepts such as VLANs, routing, dynamic protocols, and network security.

---

## 🚀 Projects Included

### 🔹 1. Office Network Design (VLAN + DHCP)
- Designed a small office network with multiple departments (HR, IT, Sales)
- Implemented VLANs for network segmentation
- Configured inter-VLAN routing (Router-on-a-Stick)
- Enabled DHCP for automatic IP address assignment

---

### 🔹 2. Secure Office Network (ACL)
- Implemented Access Control Lists (ACL) for network security
- Restricted communication between specific departments
- Tested and verified traffic filtering using ping

---

### 🔹 3. Multi-Router Network using OSPF
- Designed a network with multiple routers
- Configured OSPF for dynamic routing
- Enabled communication between different networks
- Verified routing tables and connectivity

---

## 🧠 Skills Demonstrated
- VLAN Configuration  
- Inter-VLAN Routing  
- DHCP Configuration  
- Access Control Lists (ACL)  
- OSPF (Dynamic Routing)  
- Network Design & Troubleshooting
